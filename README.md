# 🗣️insfollow
increasing Instagram Followers Tool 


# 🗣️Requirements
1. Curl
2. Openssh

# 🗣️How to INSTALL in Termux
$ pkg up -y

$ pkg install openssl-tool

$ pkg install curl

$ pkg install git

$ git clone https://github.com/kingbillhacker/insfollow

$ cd insfollow

$ chmod +x insfollow.sh

$ termux-wake-lock

$ bash insfollow.sh
